### 🔧 기술 스택

* Front-end
   - JavaScript 
   - React
   - Sass
 
* Back-end 
   - aqerytool
    - selenium
    - python
    - django
    - bcrypt
    - jwt
    - mysql
---------------------
### 👨‍👩‍👧‍👧 팀원 및 담당 역할
 
* 김우석 : 상품 리스트 페이지 구현
* 박남준 : 메인 페이지, 상품 상세 페이지, 미니 장바구니 장바구니 , 푸터
* 손종일 : 로그인, 회원가입 페이지 및 nav Bar 구현
* 프론트 엔드 공통
  - CRA 초기 세팅
  - git, github, Trello 를 통한 일정 관리
  - 백엔드 API 통신
  - Mock-Data 활용

* 김수현 : 
   - 장고 초기세팅
   - 모델링
   - 회원가입 views.py, 로그인 SignUPView, SignInview.py
   - User models.py 
* 오가빈
   - 데이터 모델링
   - 상품 모델 작성
   - 상품 리스트 뷰 페이지 작성
   - 상품 상세페이지 뷰 작성
